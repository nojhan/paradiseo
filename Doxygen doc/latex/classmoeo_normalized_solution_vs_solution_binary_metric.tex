class moeo\+Normalized\+Solution\+Vs\+Solution\+Binary\+Metric$<$ Objective\+Vector, R $>$}

Base class for binary metrics dedicated to the performance comparison between two solutions\textquotesingle{}s objective vectors using normalized values. Then, indicator values lie in the interval \mbox{[}-\/1,1\mbox{]}. Note that you have to set the bounds for every objective before using the operator().
