\doxysubsection{Detailed Description}
Defines a name (\mbox{\hyperlink{classeo_object_a1c02745db786e7bb46dea93b560fe685}{class\+Name}}\#), used when loading or saving a state.

It is recommended that you only derive from \mbox{\hyperlink{classeo_object}{eo\+Object}} in concrete classes. Some parts of \mbox{\hyperlink{class_e_o}{EO}} do not implement this yet, but that will change in the future. \mbox{\hyperlink{classeo_object}{eo\+Object}}, together with \mbox{\hyperlink{classeo_persistent}{eo\+Persistent}} and \mbox{\hyperlink{classeo_printable}{eo\+Printable}} provide a simple persistence framework that is only needed when the classes have state that changes at runtime.
