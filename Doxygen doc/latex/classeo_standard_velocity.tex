\doxysubsection{Detailed Description}
\subsubsection*{template$<$class P\+OT$>$\newline
class eo\+Standard\+Velocity$<$ P\+O\+T $>$}

Standard velocity performer for particle swarm optimization. Derivated from abstract \mbox{\hyperlink{classeo_velocity}{eo\+Velocity}}, At step t\+: v(t+1)= w $\ast$ v(t) + c1 $\ast$ r1 $\ast$ ( xbest(t)-\/x(t) ) + c2 $\ast$ r2 $\ast$ ( lbest(t) -\/ x(t) ) lbest depends on the topology evolved, when using \mbox{\hyperlink{classeo_star_topology}{eo\+Star\+Topology}}, lbest corresponds to the global. Otherwise, lbest is a \char`\"{}local best\char`\"{}, i.\+e the best in a neighborhood.
