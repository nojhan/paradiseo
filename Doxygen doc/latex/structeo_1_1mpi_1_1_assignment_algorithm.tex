\doxysubsection{Detailed Description}
Contains informations on the available workers and allows to find assignees for jobs. 

Available workers are workers who aren\textquotesingle{}t processing anything. When they\textquotesingle{}ve received an order, workers switch from the state \char`\"{}available\char`\"{} to the state \char`\"{}busy\char`\"{}, and the master has to wait for their response for considering them available again.

Indicates who are the workers which do nothing. 

At the end of the algorithm, the master has to warn all the workers that it\textquotesingle{}s done. All the workers mean, the workers which are currently processing data, and the other ones who could be waiting \+: the idles. This function indicates to the master which worker aren\textquotesingle{}t doing anything.
