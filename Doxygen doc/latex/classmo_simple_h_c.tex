class mo\+Simple\+H\+C$<$ Neighbor $>$}

Simple HC\+: Hill-\/\+Climbing local search

At each iteration, the first best solution in the neighborhood is selected if the selected neighbor have higher fitness than the current solution then the solution is replaced by the selected neighbor the algorithm stops when there is no higher neighbor
