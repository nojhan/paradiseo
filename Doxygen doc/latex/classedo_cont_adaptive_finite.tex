class edo\+Cont\+Adaptive\+Finite$<$ D $>$}

A continuator that check if any element in the parameters of an edo\+Normal\+Adaptive distribution are finite

If any element of any parameter is infinity or NaN (Not A Number), it will ask for a stop.
