class moeo\+Euclidean\+Distance$<$ M\+O\+E\+O\+T $>$}

A class allowing to compute an euclidian distance between two solutions in the objective space with normalized objective values (i.\+e. between 0 and 1). A distance value then lies between 0 and sqrt(n\+Objectives).
