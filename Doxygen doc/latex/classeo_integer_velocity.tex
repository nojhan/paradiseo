\subsubsection*{template$<$class P\+OT$>$\newline
class eo\+Integer\+Velocity$<$ P\+O\+T $>$}

Integer velocity performer for particle swarm optimization. Derivated from abstract \mbox{\hyperlink{classeo_velocity}{eo\+Velocity}}, At step t\+: v(t+1)= c1 $\ast$ v(t) + c2 $\ast$ r2 $\ast$ ( xbest(t)-\/x(t) ) + c3 $\ast$ r3 $\ast$ ( gbest(t) -\/ x(t) ) v(t) is an I\+NT for any time step (ci given and Ri chosen at random in \mbox{[}0;1\mbox{]}).
