\mbox{\hyperlink{classparadiseo_1_1smp_1_1_island_notifier}{Island\+Notifier}}\+: The notifier will perform the binded task each generation.

The island notifier makes the binded task executed by the island which observes. We can imagine that a continuator checks if we have to execute the binded task. At the moment, the task is performed each generation.
