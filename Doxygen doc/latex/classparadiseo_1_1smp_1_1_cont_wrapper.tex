\mbox{\hyperlink{classparadiseo_1_1smp_1_1_cont_wrapper}{Cont\+Wrapper}}\+: Utility class to wrap the algorithm continuators with island notifier.

Utility class to wrap the algorithm continuators with island notifier during the island construction. By using the wrapper, we do not have to modify original continuators inside the \mbox{\hyperlink{classparadiseo_1_1smp_1_1_island_model}{Island\+Model}} and then, it avoids some side effects.
