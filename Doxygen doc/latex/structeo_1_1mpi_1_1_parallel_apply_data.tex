\doxysubsection{Detailed Description}
\subsubsection*{template$<$class E\+OT$>$\newline
struct eo\+::mpi\+::\+Parallel\+Apply\+Data$<$ E\+O\+T $>$}

Data useful for a parallel apply (map). 

A parallel apply needs at least the functor to apply to every element of the table, and the table itself, whereas it can be set later with the function \mbox{\hyperlink{structeo_1_1mpi_1_1_parallel_apply_data_a2dc1d4578d47e85f8df93f115477642e}{init()}}. Master rank is also needed, to send it informations and receive informations from it, inside the functors (the job knows these values, but the functors don\textquotesingle{}t). The size of a packet can be tuned here.

Internal attributes contain\+:
\begin{DoxyItemize}
\item (useful for master) the index of the next element to be evaluated.
\item (useful for master) a map containing links between M\+PI ranks and slices of the table which the worker with this rank has evaluated. Without this map, when receiving results from a worker, the master couldn\textquotesingle{}t be able to replace the right elements in the table. 
\end{DoxyItemize}
