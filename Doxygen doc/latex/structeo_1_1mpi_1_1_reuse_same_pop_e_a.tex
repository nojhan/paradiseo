\subsubsection*{template$<$class E\+OT$>$\newline
struct eo\+::mpi\+::\+Reuse\+Same\+Pop\+E\+A$<$ E\+O\+T $>$}

For a Genetic Algorithm, reuses the same population without modifying it after a run. 

This means, if you launch a run after another one, you\textquotesingle{}ll make evolve the same population.

The evaluator should also be sent, as the population needs to be evaluated at the first time.
